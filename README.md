\# paper\_assistant\_ft



A fine-tuning project for a Chinese paper reading and experiment analysis assistant.



\## Planned stack

\- LLaMA-Factory

\- Qwen3-4B-Instruct

\- LoRA SFT



\## Project structure

\- configs/: training configs

\- scripts/: data processing / inference / evaluation scripts

\- data/: raw and processed datasets

\- outputs/: model outputs

\- logs/: training logs
